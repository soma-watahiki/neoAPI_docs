"str" neoapi.IntegerFeature.GetRepresentation (   self)

Get a value, which recommends the representation type of the Feature object in a GUI The presentation is only available for the interface types IFloat and IInteger.

Possible values are:

Linear - IFloat and IInteger
Logarithmic - IFloat and IInteger
PureNumber - IFloat and IInteger
Boolean - IInteger
HexNumber - IInteger
IPV4Address - IInteger
MACAddress - IInteger
