Detailed Description

Provides an object to get access to image data and its properties. 

 The Image class provides a set of methodes to get information about, and work with an image (such as pixelformat,
 size and offset).

◆ Convert()

"Image" neoapi.Image.Convert (   self,
*  args 
)

Convert the Image based on the provided settings. 

 A camera can stream images in different formats. However this might not be the format you need to work with
 the image data. You can use this method to convert an image from one format to another. (e.g. convert
 BayerRG8 to BGR to work with the image data using OpenCV).

 Additionally you may want to apply sharpening or want to change the color temperature. This can be achieved by providing the corresponding settings.

 Please see NeoAPI.PixelFormat for a list of possible formats.
Note
Not all input formats can be converted to all output formats. You can use the Image.GetAvailablePixelFormats() or the Image.IsPixelFormatAvailable() to check if a conversion is possible.
Parameters
[in]*argsArguments:
pixelformat The target PixelFormat name
or
settings The settings for the converted image.
Returns
A new Image with requested Settings
Exceptions
bgapiexception are covered by resulting empty image

◆ GetPixelFormat()

"str" neoapi.Image.GetPixelFormat (   self)

Get the pixel format of the image, returns an empty string if the image is empty. 

 The GenICam SFNC defines many different PixelFormats which are used to transfer image data from a camera such
 as RGB, YUV and BayerRG. Knowledge about it is essential so the image can be converted to a format needed to
 process it.
Note
More information about the GenICam SFNC (Standard Feature Naming Convention) can be found in the GenICam SFNC Document on the EMVA Website.
Returns
The pixel format of the image
