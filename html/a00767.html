Detailed Description

Provides access to events.

This class provides an easy way to work with events. It provides access to camera events as well as plug and play events.

The Class has a set of methods to get information about an event.

More information about the events of your specific camera can be found in the documentation of your camera available at the member area of the Baumer Website.
