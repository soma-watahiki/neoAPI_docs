Detailed Description

Provides access to camera features This class provides an easy way to work with camera features.

It provides access to the GenICam SFNC features as well as custom features the camera might support. The Class has a set of methods to get information about and work with a feature (such as type, minimum and maximum values), find out if a feature is accessible (available/readable/writable) and to read or write features. More information about the GenICam SFNC (standard feature naming convention) can be found at the EMVA Website. More information about the features of your specific camera can be found in the documentation of your camera available at the member area of the Baumer Website.

◆ GetInterface()

"str" neoapi.Feature.GetInterface (   self)

Get the GenICam data type of the Feature object GenICam features can have different interface (data) types.

Depending on the interface type, different feature access methods are provided

IInteger - data type integer
IFloat - data type float
IBool - data type boolean
IString - data type string
IEnumeration - data type enumeration, can be a selector or an enumeration
ICommand - command features are executable features
ICategory - used to group features which belong together
Returns
The interface type of the Feature object
Exceptions
FeatureAccessExceptionThe calling object is not valid
