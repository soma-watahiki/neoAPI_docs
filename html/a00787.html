"str" neoapi.BaseFeature.GetInterface (   self)

Get the GenICam data type of the Feature object.

GenICam features can have different interface (data) types. Depending on the interface type, different feature access methods are provided.

IInteger - data type integer
IFloat - data type float
IBool - data type boolean
IString - data type string
IEnumeration - data type enumeration, can be a selector or an enumeration
ICommand - command features are executable features
ICategory - used to group features which belong together
Returns
The interface type of the Feature object
Exceptions
FeatureAccessExceptionThe calling object is not valid
