Detailed Description

Color Transformation Matrix.

This class provides methods to configure post processing. You can pass an instance to the Convert method of an image to apply the settings.

USAGE:

matrix = neoapi.ColorMatrix() # Create a ColorMatrix object
settings = neoapi.ConverterSettings() # Create a ConverterSettings object
matrix.SetValue(neoapi.ColorMatrix.Gain_Gain00, -0.3) # Set a single value
settings.SetColorTransformationMatrix(matrix) # Set the ColorMatrix to ConverterSettings
